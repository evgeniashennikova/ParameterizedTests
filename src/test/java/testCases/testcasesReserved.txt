Заголовок: "Авторизация на сайте www.reserved.com"
Приоритет: CRITICAL
Предусловия: Открыть сайт www.reserved.com/ru/ru/

Шаги:
1. Нажать на ссылку Аккаунт.
Ожидаемый результат: осуществлён переход на страницу авторизации/регистрации.
2. Ввести адрес электронной почты.
Ожидаемый результат: в поле для ввода email отображается введённый адрес электронной почты.
3. Ввести пароль.
Ожидаемый результат: в поле для ввода пароля отображается введённый пароль.
4. Нажать кнопку "Войти".
Ожидаемый результат: отображается сообщение "Неверный логин или пароль".
